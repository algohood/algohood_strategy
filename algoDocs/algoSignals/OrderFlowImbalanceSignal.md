**信号名称**：OrderFlowImbalanceSignal

**信号概述**：
基于微观市场结构理论，通过实时订单流压力监测与自适应阈值调节，捕捉买卖动能失衡机会。融合波动率自适应机制与状态衰减因子，实现高频场景下的稳健信号生成。理论支撑包括激进做市商模型和微观噪声交易理论，预计在0.5-2秒时间框架内具有显著预测能力。

**设计模式**：
- 模式选择：半无状态
- 选择原因：
  1. 动态阈值需要持续状态跟踪
  2. O(1)空间复杂度符合高频要求
  3. 风险控制参数需要实时反馈调节

**特征提取**：
- 数据字段：price/amount/direction/recv_ts
- 特征计算：
  * 当前批次特征：
    1. 标准压力值 SPV = Σ(amount^0.8 * direction) / max(batch_duration, 0.005)
    2. 价值不对称系数 VAC = (vwap_cubic - price^3)/(3*σ_price^2 + 1e-12)
    3. 流动性熵 LE = 1 - (Σq_i^3)/(Σq_i)^3 （q_i为各价位成交量）
  * 状态变量更新：
    1. 动态阈值θ_t = 0.85*EMA_30(SPV) + 0.15*MAD_50(SPV)
    2. 压力持续计数 C_t = 衰减因子γ*C_{t-1} + I(SPV突破θ_t)
    3. 波动水平σ = 滚动50批次VAC的归一化波动率
- 复杂度分析：
  时间复杂度O(4n) 空间复杂度O(5)

**信号计算**：
- 计算特点：
  * 采用AVX2指令集并行处理浮点运算
  * 量化FFT加速熵值计算
- 信号生成：
  1. 复合信号强度 SS = tanh(SPV) * exp(-VAC) * LE^0.5
  2. 持续确认条件 CP = C_t ≥ Quantile75(window_counts) + 2
  3. 报价异常筛查 ΔBidAsk < 3*σ_TypeII
  - 核心触发规则：
    多头准入：SS > θ_t ∧ CP ∧ ΔBidAsk <1.2σ_Ask
    空头准入：SS < -θ_t ∧ CP ∧ ΔBidAsk >0.8σ_Bid

**参数说明**：
- 初始θ设定：0.7*P80(SPV_init) + 0.3*P20(SPV_init) （100批次初始化）
- 自适应窗口：Win = [Fib(n%8)+15]（斐波那契数列索引）
  - 示例：当n=5时窗口=15+5, n=7时窗口=15+13
- 动态衰减因子：γ = 1 - 1/(1+epoch^0.7) （epoch=累计触发次数）
- 调优重点：VAC多项式次数（范围2-5），LE指数敏感性（0.3-0.7）

**使用说明**：
- 适用频率：300-800ms推送周期
- 最佳场景：指数期货主力合约/加密永续合约
- 必要配合：
  - 需与订单薄失衡信号正交化处理
  - 建议组合使用冰山订单检测模块
- 风险警示：
  - 宏观数据发布前30分钟停用
  - 流动系数<0.3时自动降频
