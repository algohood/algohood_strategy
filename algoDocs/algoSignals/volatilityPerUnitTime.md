**信号名称**：单位时间波动率信号（VolatilityPerUnitTime）
**信号概述**：通过计算单位时间内的价格波动率，判断市场波动性变化。该信号旨在捕捉市场波动性的快速变化，适用于重要数据发布时段等市场波动性快速变化的场景。
**适用场景**：适用于市场波动性快速变化的场景，如重要数据发布时段。
**计算逻辑**：
1. 设置一个时间窗口（如1分钟）。
2. 在窗口内计算价格的波动率（即价格的标准差）。
3. 将波动率标准化处理，得到相对变化指标。
   - 计算窗口内波动率的均值和标准差。
   - 使用Z-score标准化方法，将当前波动率转换为标准分数。
   - 公式：Z = (X - μ) / σ，其中X为当前波动率，μ为窗口内波动率的均值，σ为窗口内波动率的标准差。
4. 根据波动率的变化趋势，判断市场波动性的变化。
**参数说明**：
- timeWindow（float，默认值60.0）：时间窗口长度（秒），取值范围（1.0, 600.0）。
- normalizationFactor（float，默认值1.0）：标准化因子，用于调整波动率的敏感度，取值范围（0.1, 10.0）。
**使用说明**：
1. 实时接收逐笔成交数据。
2. 在设定的时间窗口内计算价格波动率。
3. 根据波动率的变化趋势，判断市场波动性的变化。
4. 最终给出交易方向：波动率上升时，建议做多；波动率下降时，建议做空。
**方案对比**：
- 优点：能及时捕捉市场波动性变化，适用于快速变化的市场环境。
- 缺点：对数据延迟敏感，需确保数据实时性。
**信号类型**：周期评估型，基于时间窗口内的数据计算波动率。
**数据处理**：需确保数据按recv_ts严格递增顺序推送，避免数据延迟影响信号准确性。
