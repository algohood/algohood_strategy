**因子名称**：PriceFluctuation
**版本**：V2

**因子概述**：
PriceFluctuation 是一个有状态的因子，通过分析一定时间窗口内的价格波动情况，识别市场的短期波动模式。该因子用于评估市场风险或捕捉波动性交易机会，帮助交易者在市场波动加剧或减弱时做出相应的交易决策。因子的理论基础是波动率理论，认为价格波动是市场风险的重要度量，波动率的变化可以反映市场情绪的变化。因子值范围为[0, 1]，值越大表示市场波动越剧烈。

**设计模式**：
- 模式选择：有状态
- 选择原因：有状态模式允许维护历史价格窗口，适合中频交易场景，能够更好地捕捉市场的短期波动特征。

**特征提取**：
- 数据字段：symbol, recv_ts, price
- 特征计算（当前批次）：
  * 当前批次特征：获取当前批次的最新价格。
  * 状态变量更新：更新历史价格窗口。
  * 历史特征窗口：
    - 窗口数据结构：collections.deque
    - 窗口大小：窗口大小 = ⌈窗口时长 / 推送频率⌉
    - 维护方式：将最新的价格加入deque的右侧，若deque的长度超过maxlen，则从左侧移除最旧的价格。
- 复杂度分析：O(n)，线性复杂度，确保实时计算。

**因子计算**：
- 计算特点：基于特征窗口分析，计算复杂度较高，延迟相对较大。
- 因子生成：
  * 历史特征窗口分析方法：
    1. 计算窗口内价格的均值和标准差：
       - 均值 = sum(价格) / 窗口大小，单位为计价币种单位（如USDT）
       - 标准差 = sqrt(sum((价格 - 均值)^2) / 窗口大小)，单位为计价币种单位（如USDT）
    2. 计算当前价格与均值的差异和标准差的比值：
       - 波动比 = (当前价格 - 均值) / 标准差
  * 归一化处理：使用指数加权方式平滑处理，抑制噪声。
    - 平滑后的因子值 = 平滑系数 * 前一因子值 + (1 - 平滑系数) * 当前因子值
  * 计算结果规范：因子值 = sigmoid(波动比)，具体形式为 sigmoid(x) = 1 / (1 + exp(-x))，范围为[0, 1]。

**参数说明**：
- 阈值参数：窗口时长（默认300秒，适用于5分钟级别的波动分析）
- 模式参数：平滑系数（默认0.9，用于指数加权平滑）
- 调优建议：根据市场波动特性调整窗口时长和平滑系数。

**使用说明**：
- 推送频率：1min
- 应用场景：中频交易，波动率预测。
- 注意事项：该因子需要较大的历史窗口来准确反映市场波动特征，适用于波动性交易策略。
