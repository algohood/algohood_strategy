**因子名称**：VolumeAcceleration
**版本**：V1

**因子概述**：本因子通过测算成交量变化的二阶导数，捕捉资金流动的突变信号。基于"量在价先"的市场共识，当成交量呈现加速增长时可能预示行情启动，适用于高频趋势突破策略。理论依据包括市场微观结构中的信息不对称理论和动能效应。

**设计模式**：
- 模式选择：半无状态
- 选择原因：
  1. 需要持续跟踪成交量的动量变化，必须维护有限状态变量
  2. 采用指数更新机制，不需要保存完整历史数据
  3. 适合10s级高频交易场景，复杂度控制在O(1)

**特征提取**：
- 数据字段：amount,direction
- 特征计算（当前批次）：
  * 当前批次特征：
    1. 方向调整成交额：Σ(amount*direction)
    2. 分笔持续时间：max(recv_ts) - min(recv_ts)
  * 状态变量更新：
    - S (动量累积量)：S_new = λ_s*S_old + (1-λ_s)*特征1
    - M (移动均值)：M_new = λ_m*M_old + (1-λ_m)*特征1
    - T (时间基量)：T_new = α*T_old + (1-α)*特征2
  * 历史特征窗口：无
- 复杂度分析：
  - 单次更新计算量：3次乘加运算，2次指数运算
  - 内存消耗：3个float型状态变量
  - 时间复杂度：O(1)

**因子计算**：
- 计算特点：
  - 每批次处理延迟≤50μs
  - CPU占用率<0.1%
- 因子生成：
  * 特征组合：
    加速度因子 = [S_new - 2*(M_new) + (λ_s^2)/(1-λ_s)*S_old] / (T_new+ε)
  * 归一化处理：
    - 使用自适应动态阈值法：当前值 / 历史90分位值
    - 衰减系数β=0.999，在线更新分位估计值

**参数说明**：
- 阈值参数：
  1. λ_s (动量衰减率)：
     - 范围：0.8-0.99
     - 默认：0.95
     - 含义：越大动量延续性越强
  2. λ_m (均值衰减率)：
     - 范围：0.9-0.999 
     - 默认：0.99
     - 含义：越大基线越稳定
  3. ε (平滑项)：1e-6
- 模式参数：
  - 时间衰减系数α=0.9
- 调优建议：
  - 市场震荡期：增大λ_m到0.998
  - 趋势市场：降低λ_s到0.85

**使用说明**：
- 推送频率：1秒级(高频场景)
- 应用场景：
  1. 突破确认：当因子值超过历史分位的3σ时
  2. 量价背离检测：结合价格方向过滤信号
- 注意事项：
  - 需配合成交量规模过滤，避免小单干扰
  - 在集合竞价时段效果减弱
  - 需定期重置分位估计值(建议每日开盘时)
